# Glossary

TLC (Total Landed Cost)
- Sum of CIF value, duties/taxes, fees, clearance, and logistics to delivery.

Delivery Point
- The point at which risk transfers to the customer (per policy).

Shipping Method
- Transport mode/terms (air/sea/land/express) with base characteristics.

Shipping Route
- Origin/destination pair and path metadata (e.g., transit points).
