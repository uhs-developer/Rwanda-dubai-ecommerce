About Us Page Independent Styling Demo

🎯 How It Works

This system provides completely independent styling for the About Us page without affecting other pages like the homepage.

Key Concept: All About Us styles are scoped with .page-about-us class, ensuring they only apply to that specific page.

🔧 Implementation

1. Body Classes Added Automatically

// In functions.php function dubaidirect_page_specific_body_classes($classes) { if (is_page('about-us') || is_page_template('page-about-us.php')) { $classes[] = 'page-about-us'; $classes[] = 'about-us-template'; } return $classes; }

2. Conditional CSS Loading

// About Us styles only load when needed if (is_page('about-us') || is_page_template('page-about-us.php')) { wp_enqueue_style( 'dubaidirect-about-us', get_template_directory_uri() . '/assets/css/about-us.css', array('dubaidirect-style'), _S_VERSION ); }

3. Scoped CSS Classes

/* about-us.css - All styles are scoped */ .page-about-us .page-hero { height: 80vh; min-height: 600px; } .page-about-us .hero-title { font-size: clamp(2.5rem, 5vw, 4rem); } /* These styles will NEVER affect other pages */

✅ Benefits

Complete Isolation: About Us styles cannot affect other pages
Performance: CSS is only loaded when needed
Maintainability: Easy to modify without risk
Scalability: Simple pattern for other pages
